The quadratic formula

The famous quadratic formula, in some countries, like Brazil and France, called as the "Bháskhara Formula",
has it origin in the XVII century. It was created by a prodigiuos Indian mathematician, called Bhaskara Akaria.
His formula is actually really simple and it's not the only way to solve a second degree polynomial equation!
But, here, we are going to learn how was it discovered, and what all the meanings of the things in a quadratic equation.


Demonstration

         ax² + bx + c = 0

Divide both sides by a :

         ax²/a + bx/a + c/a = 0/a

ax²/a = x² :

         ² + bx/a + c/a = 0

Add, b²/2a on both sides, so you can get the (a + b)² case :

          x² + (b/a)x + (b/2a)² - (b/2a)² - (b/2a)² + c/a = 0

          (x + b/2a)² + (4ac/4a² - b²/4a) = 0

          (x + b/2a)² = - (4ac/4a² - b²/4a)

          x + b/2a = sqrt(b² - 4ac/4a²)

          x = -b/2a ± sqrt(b² - 4ac/4a²)

          x = -b/2a ± sqrt(b² - 4ac)/2a

Let (b² - 4ac) = Δ :

        x = -b/2a ± sqrt(Δ)/2a

The discriminant

The value of the discriminant Δ has a meaningful paper in a quadratic equation. The value of the discriminant can tell if the function has 0, 1 or 2 real roots!

  • If b² - 4ac is a positive real nuymber, the function has 2 real roots.
  • If b² - 4ac is equal to 0, so the funtion has 2 equal real roots, or, just one real root.
  • If b² - 4ac is equal to a negative real number, so the funtion has no real roots, once there isn't in the real numbers, a square root of a negative number.


The 'a' coefficient

The value that is multiplying the x² in a quadratic formula is called 'a coefficient', and determines the slope of the parabola.
So, for bigger values of 'a' the parabola is more "closed" than for smaller values. So then, for bigger values, the slope of the
parabola is bigger than for smaller values!

*Note that the slope of the [f(x) = 10x²] equation is much bigger than the [f(x) = x²], just because the 'a' coefficient is much bigger!

Conclusion

We can conclude about the 'a' coefficient that:

  1. It's the value that multiplies the x² on a quadratic.
  2. It determines the slope of the parabola.
  3. If it's equal to 0, there is no quadratic equation.

The 'b' coefficient

The value that is multiplying the x in a quadratic formula is called 'b coefficient', and represents the angular coefficient of the the linear equation that tangents the parabola in (x=0).
Using the image below to ilustrate it, we can see that the linear equation that tangents the function [f(x) = ax² + bx + c] is:

          f(x) = bx + c

Conclusion

We can conclude about the 'b' coefficient that:

  1. It's the value that multiplies x in the quadratic.
  2. It determines the angular coefficient of the tangent linear equation of a quadratic.

The 'c' coefficient

The value that isn't multiplying anything in a quadratic formula is called 'c coefficient', and determines where, in the Y axis, the parabola will touch.
It happens, because:

        f(0) = (0 * a) + (0 * b) + c
      => f(0) = c

So: when x = 0, f(x) = c.

  3. If there is no 'c' coefficient, then the parabola will touch the y axis in the origin (0,0)
